# **App Name**: PrestaControl

## Core Features:

- Loan Application: Allow users to submit loan applications with required details such as loan amount, purpose, and repayment terms.
- Loan Approval Workflow: Implement a workflow for approving or rejecting loan applications, including notifications to the applicant.
- Repayment Schedule: Automatically generate a repayment schedule based on the loan terms, including due dates and amounts.
- Payment Tracking: Track payments made by borrowers and automatically update loan balances.
- Late Payment Reminders: Send automated reminders to borrowers who have missed payments.
- Reporting and Analytics: Generate reports on loan performance, including outstanding balances, delinquency rates, and overall profitability.

## Style Guidelines:

- Primary color: Use a deep navy blue (#1E3A8A) to convey trust and stability in financial matters.
- Background color: Very light blue-gray (#F0F4F8), providing a clean and professional backdrop.
- Accent color: Muted teal (#38A3A5), complementing the primary blue with a hint of freshness and modernity, especially for interactive elements.
- Body and headline font: 'Inter', a sans-serif, for a clean and modern user interface as per user request.
- Use simple, outlined icons from a consistent set to represent various features and actions, maintaining a clean and intuitive interface.
- Employ a grid-based layout for responsiveness, ensuring optimal viewing and interaction on all devices. The design is responsive because the prompt includes @tailwind base; @tailwind components; @tailwind utilities;
- Incorporate subtle transitions and animations to provide feedback on user interactions and guide users through workflows smoothly.
